#  Installation

Ce projet utilise **Node.js**, **Express**, **Better-SQLite3**, **CORS** et **JSON Web Tokens**.

---

## Installation des dépendances

Exécutez la commande suivante dans le dossier du projet :

```bash
npm install express better-sqlite3 cors jsonwebtoken
```

---

## Version de Node.js

Le projet nécessite **Node.js v20**.  
Avec **nvm**, vous pouvez activer cette version :

```bash
nvm use 20
```

Si vous ne l’avez pas encore installée :

```bash
nvm install 20
```

---

## ▶Lancer le projet

Pour démarrer le serveur :

```bash
node app.js
```
